HTTP Web Server

Folder consist of:
-WebServer.py - multithreaded web server
-/webpage/index.html - webpage
-/webpage/fallTree.jpg - picture for index.html
-WebClient.py - HTTP client script

How to compile server:
-Start up terminal, open directory and type in command:
	python WebServer.py
-Go to web browser and type in url in browser to see webpage:
	http://localhost:8080
